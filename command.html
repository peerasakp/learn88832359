คำสั่ง

ให้นิสิตใช้ git สำหรับฝึกฝนการใช้งานระบบควบคุมเวอร์ชัน

  1. clone ซอร์สโค้ดจาก github
  2. แต่ละทีมสร้าง branch สำหรับพัฒนางานของตนเอง เช่น dev_team1
  3. แต่ละทีมสร้างเมนู (ด้านซ้าย) เพื่อลิงก์ไปยังหน้าข้อมูลทีม โดยระบุชื่อไฟล์ คือ   team หมายเลข.html เช่น  team1.html
  4. หน้าข้อมูลทีม แสดงรายชื่อสมาชิก ตำแหน่ง และลิงก์ไปยังหน้าโปรไฟล์ของตนเอง (แต่ละคนใส่ชื่อตนเอง)
  5. สมาชิกแต่ละคนสร้างหน้าโปรไฟล์ของตนเอง โดยระบุชื่อไฟล์ คือ รหัสนิสิต.html เช่น 60160001.html
  6. เมื่อพัฒนาเสร็จแล้ว ให้รวมเข้ากับ master branch
